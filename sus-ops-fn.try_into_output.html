template <class T>
auto try_into_output(T&& t) -> sus::ops::TryImpl<std::remove_cvref_t<T>>::Output
requires 
sus::ops::Try<T>
sus::mem::IsMoveRef<decltype(t)>

Unwraps from the Try type that is currently in its success state (is_success() would return true) to produce its success value.

For instance, this unwraps an Result<T, E> which can represent success or failure into T which represents only success in the type system.

Panics

If the input is not in a success state (is_success() would return true) the function will panic.
