Compares two options.

  • Satisfies StrongOrd<Option<T>> if StrongOrd<T>.
  • Satisfies Ord<Option<T>> if Ord<T>.
  • Satisfies PartialOrd<Option<T>> if PartialOrd<T>.

The non-template overloads allow some/none marker types to convert to an option for comparison.
