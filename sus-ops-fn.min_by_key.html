Returns the element that gives the minimum value from the specified function.

Returns the first argument if the comparison determines them to be equal.

By default this receives and returns objects by value. To receive and return references, specify the type parameter, such as: sus::ops::min_by_key<i32&>(a, b, k).

Note that if either input is a temporary object this can return a reference to an object past its lifetime.
