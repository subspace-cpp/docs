Function Subspace :: sus :: construct :: move_into
template <class FromType>
automove_into(FromType & from) -> auto
requires 
!std::is_const_v<std::remove_reference_t<FromType>>
!std::is_array_v<FromType>
template <class FromType>
automove_into(FromType && from) -> auto
requires 
std::is_rvalue_reference_v<FromType &&>
!std::is_const_v<std::remove_reference_t<FromType>>
!std::is_array_v<FromType>

Moves from and converts from the given value to whatever a receiver requires.

This is sugar for sus::into(sus::move(x)), which can be written as sus::move_into(x). The result will be receivable if Into<FromType, ToType> is satisfied.

To capture an rvalue, use into. Note that arrays can not be moved, so can not be given to move_into.
