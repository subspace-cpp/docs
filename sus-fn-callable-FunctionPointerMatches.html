Concept Subspace :: sus :: fn :: callable :: FunctionPointerMatches 
template <class T, class R, class... Args>
concept FunctionPointerMatches
requires
sus::fn::callable::FunctionPointer < T > 
requires (R(*p)(Args...), T t) {
        { p = t };
    }

Verifies that T is a function pointer (or captureless lambda) that receives exactly Args as its parameters without conversion, and returns R without conversion.

This is concept is useful if you intend to store the pointer in a strongly typed function pointer, as the types must match exactly. If you only intend to call the function pointer, prefer FunctionPointerReturns which allows appropriate conversions.
