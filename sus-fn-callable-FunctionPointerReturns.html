concept
FunctionPointerReturns
Concept Subspace :: sus :: fn :: callable :: FunctionPointerReturns
template <class T, class R, class... Args>
concept FunctionPointerReturns
requires
sus::fn::callable::FunctionPointer < T >
std::convertible_to < std::invoke_result_t<T, Args...> R > 
requires (R(*p)(Args...), T t) {
        { p = t };
    }

Verifies that T is a function pointer (or captureless lambda) that returns a type convertible to R when called with Args.

This concept allows conversion of Args to the function's actual receiving types and conversion from the function's return type to R.
