Macro Subspace :: sus_eval_macro
source
#define sus_eval_macro(x, ...)

A macro that runs a macro against any arguments.

This is useful to join together macros, for instance to run one macro against the output of another macro.

Example

#define f(s) hello(s)
sus_eval_macro(f, "world")
// Evaluates to: `hello("world")`
