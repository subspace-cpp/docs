Function Subspace :: sus :: iter :: once
template <class Item>
autoonce(Option<Item> o) -> Once<Item>
template <class Item>
autoonce(Option<Item> o) -> Once<Item>

Constructs a Once iterator that will return o and then None. If o is None, then the iterator will be empty on construction.

Example

auto o = sus::iter::once<u16>(sus::some(3_u16));
sus::check(o.next().unwrap() == 3_u16);
