Function Subspace :: sus :: ops :: min
template <class T>
automin(T a, T b) -> T
requires
sus::ops::Ord < T >

Compares and returns the minimum of two values.

Returns the first argument if the comparison determines them to be equal.

By default this receives and returns objects by value. To receive and return references, specify the type parameter, such as: sus::ops::min<i32&>(a, b).

Note that if either input is a temporary object this can return a reference to an object past its lifetime.
