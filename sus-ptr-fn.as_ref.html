template <class T>
auto as_ref(T* pointer) -> Option<T&>

Returns an empty Option<T&> if the T* pointer is null, or else returns a reference to the value T wrapped in an Option<T&>.

This is the pointer::as_ref operation in Rust. However it is not unsafe in C++ as no lifetime is created as part of the operation.
