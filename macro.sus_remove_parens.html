Macro Subspace :: sus_remove_parens
#define sus_remove_parens(x)

Remove parentheses around one or more arguments, if they are present.

It performs the following transformations, for any number of values inside the parens.

  • x => x
  • (x) => x
  • (x, y) => x, y
