Classes
  • NonNull

    A pointer wrapper which holds a never-null pointer.

Functions
  • copy

    Copies count * size_of<T>() bytes from src to dst. The source and destination may overlap.

  • copy_nonoverlapping

    Copies count * size_of<T>() bytes from src to dst. The source and destination must not overlap.

  • swap_nonoverlapping

    Swaps the object array at x with the object array at y, where both arrays have a length of count.

Operators
  • operator<=>

    Satisfies the StrongOrd<NonNull<T>> concept if the pointers are comparable and thus satisfy StrongOrd as well.

  • operator==

    Satisfies the Eq<NonNull<T>, NonNull<U>> concept if the pointers are comparable and thus satisfy Eq as well.
