Concept Subspace :: sus :: ops :: ExclusiveStrongOrd
template <class Lhs, class Rhs = Lhs>
concept ExclusiveStrongOrd
requires
sus::ops::StrongOrd < Lhs Rhs >

Concept for types that have a unique total ordering (aka std::strong_ordering).

This is an alias for StrongOrd, but exists as a set with ExclusiveOrd and ExclusivePartialOrd.
